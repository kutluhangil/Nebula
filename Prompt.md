# COSMOS — Planet Intelligence Dashboard

## Project Vision

Build a premium, modern, AI-powered web application that combines live space exploration, astronomy, Earth events, and scientific discoveries into one immersive experience.

This is **NOT** a news website.

This should feel like a futuristic scientific operating system where users can explore everything happening on Earth and in Space in real time.

The goal is to create one of the most beautiful science dashboards on the internet.
